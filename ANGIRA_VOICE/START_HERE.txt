   - Streaming support planned

3. QUERY PROCESSING
   - Verbalizes mathematical expressions
   - Normalizes shorthand notation
   - Converts symbols to natural language
   - Example: √x² dx → "square root of x squared dx"

4. INTENT CLASSIFICATION
   - Rule-based simple/complex classification
   - Analyzes query length
   - Detects complex keywords
   - Counts mathematical operators
   - Returns: "simple" or "complex"

5. INTELLIGENT REASONING
   - Routes complex queries to GPT-4
   - GPT used ONLY for reasoning
   - Never for speech generation
   - Cost-optimized routing

6. SMART RESPONSE ROUTING
   - Simple queries: Full text-to-speech
   - Complex queries: Brief acknowledgment TTS
   - Minimizes API costs
   - Maximizes user satisfaction

7. TEXT-TO-SPEECH (Gemini Flash 2.5 Live)
   - Converts text to speech
   - Placeholder implementation
   - Cost-optimized output
   - 2000 character limit for simple queries
